Welcome to the First Aid Kit IoT Project

This project utilizes an Arduino ESP32 board to collect data from various sensors in a first aid kit, including heartbeat, temperature, pressure, humidity, thermal, and motion sensors.

Latest Sensor Data

You can view the current sensor readings on the Sensor Data page.

User Registration

Users can register their ID cards to access the first aid kit. When a registered user scans their card, the website will display their name, student information, program, and age.
